# Receiver Configuration

## Register your address

Register your address to receive receipts.

## Register company handles

Register the companies that you're authorized to receive receipts for.

Add —

```bash
curl --request POST \
  --url 'https://registry.versa.org/handle' \
  --header 'Authorization: <CLIENT_ID:CLIENT_SECRET>' \
  --header 'Content-Type: application/json' \
  --data '{
    "handle_type": "email_domain",
    "handle": "acme.com"
  }'
```

Or remove —

```bash
curl --request DELETE \
  --url 'https://registry.versa.org/handle' \
  --header 'Authorization: <CLIENT_ID:CLIENT_SECRET>' \
  --header 'Content-Type: application/json' \
  --data '{
    "handle_type": "email_domain",
    "handle": "acme.com"
  }'
```
