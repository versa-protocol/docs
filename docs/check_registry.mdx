# Check Registry

It may be useful to see what registered receivers exist for a given customer.

The 'check_registry' endpoint expects a body containing the transaction handles
for a hypothetical receipt registration and returns a list of matched receiver profiles.

Request:

```bash
curl --request POST \
  --url 'https://registry.versa.org/check_registry' \
  --header 'Authorization: Basic CLIENT_ID:CLIENT_SECRET' \
  --header 'Content-Type: application/json' \
  --data '{
    "customer_email_domain": "acme.com"
  }'
```

Returns:

```json
{
  "mode": "prod",
  "receivers": [
    {
      "org": {
        "id": "org_67b09701789b44428e69664e7f1eb165",
        "name": "Acme Bank",
        "slug": "acme-bank",
        ...
      },
      "managed_by": "receiver"
    }
  ]
}
```
