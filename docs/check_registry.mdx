# Check Registry

It may be useful to see what registered receivers exist for a given user.

The 'regcheck' endpoint returns a list of matched receiver profiles.

Request:

```bash
curl --request POST \
  --url 'https://registry.versa.org/regcheck' \
  --header 'Authorization: Basic CLIENT_ID:CLIENT_SECRET' \
  --header 'Content-Type: application/json' \
  --data '{
    "customer_email_domain": "acme.com"
  }'
```

Returns:

```json
{
  "env": "prod",
  "receivers": [
    {
      "client_id": "versa_cid_05d108be2b974577b203a4d907a61e7d,
      "receiver": {
        "id": "org_67b09701789b44428e69664e7f1eb165",
        "name": "Acme Bank",
        "slug": "acme-bank",
        ...
      }
    }
  ]
}
```
