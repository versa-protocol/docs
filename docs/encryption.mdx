# Encryption

A list of encryption tools for use with the Versa network.

## AES-SIV

- Rust: [aes-siv](https://crates.io/crates/aes-siv)
- Python: [pycryptodome](https://pycryptodome.readthedocs.io/en/latest/src/cipher/aes.html)
- Go: [crypto/aes](https://pkg.go.dev/crypto/aes)
- JavaScript (Node.js): [aes-js](https://www.npmjs.com/package/aes-js)
